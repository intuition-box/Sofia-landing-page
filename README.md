# Sofia Chronicles

Development blog documenting the creation of Sofia, an AI built as part of an educational project. This site transparently documents the development process, challenges encountered, and project evolution.

## About

Sofia Chronicles is a "build in public" blog that openly shares the journey of creating an artificial intelligence. The goal is to document every step of the development process to provide a transparent view of the creative and technical journey.

The site is built with Docusaurus v3 to offer an optimal reading experience and intuitive navigation through the different phases of the project.
