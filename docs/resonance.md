# Resonance

Resonance is Sofia's social layer — discover what others are certifying, vote on content, and participate in community debates.

## Circle Feed

See real-time certifications from your Trust Circle.

### Features

| Feature | Description |
|---------|-------------|
| **Real-time feed** | Certifications from people you follow/trust |
| **Vote Like/Dislike** | React to certifications (1 TRUST per vote) |
| **Filter by intention** | Show only Work, Learning, Fun, etc. |
| **Visit profiles** | See other users' full certification history |

### Feed Entry

```
┌─────────────────────────────────────────┐
│  alice.eth certified github.com/repo    │
│  for Learning • 2 hours ago             │
│                                         │
│  Deposit: 0.5 TRUST | Explorer          │
│                                         │
│  [👍 Like] [👎 Dislike] [View Profile]  │
└─────────────────────────────────────────┘
```

## Vote (Debate)

Participate in community debates by supporting or opposing claims.

### Sofia Claims

**Sofia Claims** are community debates curated by Sofia, displayed as an interactive card deck.

```
┌─────────────────────────────────────────┐
│                                         │
│  "AI will replace most coding jobs      │
│   within 5 years"                       │
│                                         │
│  ████████████░░░░░░░░  Support: 67%     │
│  ░░░░░░░░░░░░████████  Oppose: 33%      │
│                                         │
│  Total staked: 1,234 TRUST              │
│                                         │
│  [Support]  [Oppose]  [Skip →]          │
│                                         │
└─────────────────────────────────────────┘
        ┌───┐ ┌───┐ ┌───┐
        │   │ │   │ │   │  ← More claims
        └───┘ └───┘ └───┘
```

### Featured Claims

**Featured Claims** are notable claims from the broader Intuition community:

- "Claude is better than ChatGPT"
- "Pineapple belongs on Pizza"
- "Bitcoin will reach $1M"

### How Voting Works

| Action | Description | Cost |
|--------|-------------|------|
| **Support** | Stake TRUST to agree with the claim | Custom TRUST |
| **Oppose** | Stake TRUST to disagree with the claim | Custom TRUST |
| **Choose curve** | Linear or Progressive staking | — |

### Battle Bar

The visual battle bar shows Support vs Oppose percentages in real-time, updating as users stake.

## Blockchain Costs

| Action | Cost |
|--------|------|
| **Like/Dislike** certification | 1 TRUST |
| **Support/Oppose** claim | Custom TRUST |
| **Support/Oppose** list entry | Custom TRUST |

All votes are **on-chain** and earn you **+5 Gold** (max 10 votes/day).

---

:::tip Earning Through Voting
Daily voting is one of the easiest ways to earn Gold consistently. Vote on 10 certifications or claims each day for 50 Gold!
:::
