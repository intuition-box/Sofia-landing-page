# AI Features

Sofia integrates AI capabilities to help you extract insights from your browsing and certifications.

## Pulse Analysis

**Pulse Analysis** lets you analyze all your currently open browser tabs and extract potential signals (triplets) to certify.

### How It Works

```
┌─────────────────────────────────────────┐
│  1. Click "Pulse Analysis"              │
│              ↓                          │
│  2. AI scans all open tabs              │
│              ↓                          │
│  3. Extracts keywords & patterns        │
│              ↓                          │
│  4. Generates potential signals         │
│              ↓                          │
│  5. You select which to amplify         │
└─────────────────────────────────────────┘
```

### What You Can Do

| Action | Description |
|--------|-------------|
| **Launch analysis** | AI analyzes all your open tabs |
| **View signals** | See extracted triplets by session |
| **Select signals** | Choose individually or in bulk |
| **Amplify signals** | Publish selected signals on-chain |
| **Delete signals** | Remove irrelevant signals or entire sessions |

### Example Output

```
┌─────────────────────────────────────────┐
│  Pulse Session — Mar 10, 2025           │
│                                         │
│  Detected theme: React Development      │
│                                         │
│  ☑ react.dev/learn          → Learning  │
│  ☑ github.com/facebook/react → Work     │
│  ☐ twitter.com/home         → Fun       │
│  ☑ stackoverflow.com/...    → Learning  │
│                                         │
│  [Amplify Selected] [Clear Session]     │
└─────────────────────────────────────────┘
```

## Interest Analysis

**Interest Analysis** categorizes your on-chain certifications into interest areas, building your expertise profile.

### How It Works

The AI analyzes your **existing certifications** (not your browsing) to identify patterns and categorize your interests.

```
Your On-Chain Certifications
           ↓
    AI Categorization
           ↓
   Interest Profile
```

### Interest Categories

| Category | Based On |
|----------|----------|
| **Software Development** | GitHub, docs, Stack Overflow certifications |
| **Design & UI/UX** | Dribbble, Figma, design tool certifications |
| **Business & Finance** | Trading, news, finance site certifications |
| **Entertainment** | YouTube, Twitch, gaming certifications |
| **Music** | Spotify, SoundCloud certifications |
| **Education** | Course platforms, tutorials certifications |

### Interest Profile View

```
┌─────────────────────────────────────────┐
│  Your Interests                         │
│                                         │
│  Software Development  ████████░░ 80%   │
│  └─ Work: 45% | Learning: 35%           │
│                                         │
│  Design & UI/UX       ████░░░░░░ 40%    │
│  └─ Inspiration: 30% | Learning: 10%    │
│                                         │
│  [Share on X/Twitter]                   │
└─────────────────────────────────────────┘
```

### Sharing

You can share your interest profile on X/Twitter with an auto-generated OG image.

## Chat with Sofia

Sofia includes a conversational AI assistant that knows about your activity and can help you.

### What Sofia Can Do

| Capability | Example |
|------------|---------|
| **Answer questions** | "What have I certified for Learning?" |
| **Provide recommendations** | "What should I read next?" |
| **Explain features** | "How do streaks work?" |
| **Help navigate** | "Show me my Trust Circle" |

### Example Conversation

```
You: What topics have I been learning about?

Sofia: Based on your Learning certifications, you've focused on:
• TypeScript (12 certifications)
• React hooks (8 certifications)
• GraphQL (5 certifications)

Your most active learning domain is github.com with 15 pages certified.
```

## Level Up Predicate Generation

When you level up a domain, the AI generates a **unique predicate** for your group, making your relationship with that domain more personalized.

### How It Works

1. You reach the certification threshold for a domain
2. You spend Gold to level up
3. AI generates a custom predicate for your group
4. The new predicate is used when you **Amplify** the domain

### Example

```
Before level up: "visits for learning"
After level up:  "studies deeply" (AI-generated)
```
